# WebHash

Next.js + MUI single-page web app for computing file hashes (MD5, SHA-1, SHA-256, SHA-512) directly in the browser.

## Features

- Drag & drop one or more files
- Multi-file selection via file picker
- Client-side hashing (no files are uploaded to a server)
- MD5, SHA-1, SHA-256, SHA-512 displayed in a responsive table
- Export results as CSV
- Light, Night (default), and Dark themes
- MUI-based responsive UI

## Getting started

```bash
npm install
npm run dev
# or
pnpm install
pnpm dev
```

Then open http://localhost:3000 in your browser.
